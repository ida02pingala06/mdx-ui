import { Steps, Step } from "@/components/mdx/steps"

## Installation

```bash
npx mdx-ui add steps
```

## Usage

```tsx
import { Steps, Step } from "@/components/mdx/steps"

<Steps>
  <Step>First step</Step>
  <Step>Second step</Step>
  <Step>Third step</Step>
</Steps>
```

### Steps with Rich Content

You can include any content inside steps, including code blocks and lists.

<div className="preview">
  <Steps>
    <Step>
      **Install dependencies**

      Run the following command:

      ```bash
      npm install mdx-ui
      ```
    </Step>
    <Step>
      **Configure your project**

      Update your configuration files:
      - Add Tailwind CSS
      - Configure MDX
      - Set up TypeScript
    </Step>
    <Step>
      **Start developing**

      You're ready to go!
    </Step>
  </Steps>
</div>

## Props

### Steps

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `children` | `React.ReactNode` | - | Step components |
| `className` | `string` | - | Additional CSS classes |

### Step

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `children` | `React.ReactNode` | - | The step content |
| `className` | `string` | - | Additional CSS classes |
